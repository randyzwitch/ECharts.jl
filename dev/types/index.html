Main Chart Types

ECharts.EChart — Type
EChart

The main chart struct returned by all ECharts.jl chart constructors (e.g. bar, line, scatter).

Fields prefixed with ec_ are ECharts.jl-internal and do not affect the JavaScript output:

  • ec_width / ec_height : canvas size in pixels (default 800 × 400)
  • ec_renderer : "canvas" (default) or "svg"
  • ec_charttype : internal label for the chart type
  • theme : the Theme applied at render time

All other fields map directly to Apache ECharts option keys (title, xAxis, yAxis, series, legend, etc.) and can be mutated after construction for fine-grained control.

See the Apache ECharts documentation for field details.

source
ECharts.colorpalettes — Constant
colorpalettes

A merged dictionary of named color palettes from ColorBrewer.jl and NoveltyColors.jl. Pass a key from this dict to colorscheme! to apply a named palette to a chart.

source

Theme

ECharts.Theme — Type
Theme

ECharts visual theme applied to a chart. All fields mirror the top-level keys of an ECharts JSON theme file. Pre-built themes exported by ECharts.jl: chalk, dark, essos, halloween, infographic, macarons, purplepassion, roma, shine, vintage, walden, westeros, wonderland, grayscale.

Pass a theme to any chart constructor via the theme keyword, or change it afterwards by setting ec.theme = mytheme.

See also theme!.

ECharts.CustomSeries — Type
CustomSeries

ECharts series configuration for custom-rendered charts. The renderItem field accepts a JSON.JSONText-wrapped JavaScript function string that ECharts calls for each data point to produce SVG/canvas primitives (rect, circle, etc.).
