heatmap

ECharts.heatmap — Function
heatmap(data)

Creates an EChart heatmap.

Methods

heatmap(h::StatsBase.Histogram)

Arguments

  • legend::Bool = false : display legend?
  • show::Bool = true : show slider denoting color range
  • calculable::Bool = true : allow user to slide endpoints to change data displayed
  • kwargs : varargs to set any field of resulting EChart struct

Notes

Reasonable defaults set for different methods of area, such as displaying a legend when two or more series present.

source
using ECharts, StatsBase, Random
Random.seed!(42)
# Simulate height (cm) and weight (kg) for 10,000 people
height = 170 .+ 10 .* randn(10000)
weight =  70 .+ 15 .* randn(10000)
h = fit(Histogram, (height, weight), nbins = 25)
ec = heatmap(h)
title!(ec, text = "Height vs Weight Distribution", subtext = "10,000 simulated subjects")
