Utility Functions

ECharts.from_json — Function
from_json(json_str; width=800, height=450, renderer="canvas", theme=roma) -> EChart

Parse an Apache ECharts JSON option string into an EChart struct.

The resulting chart can be rendered, modified with the standard ECharts.jl modifier functions, and re-exported.

Any JSON keys that have no corresponding field in the Julia type system emit a @warn message so that gaps can be identified and fixed. Pass warn_unknown=false to suppress these warnings.

source
ECharts.savefig — Function
savefig(filename::AbstractString, ec::Union{EChart, EChartRaw})

Save an EChart to a file. The output format is determined by the file extension:

  • .html — self-contained HTML page with ECharts.js embedded
  • .json — raw ECharts option JSON (the object passed to setOption)
  • .svg — server-side rendered SVG via NodeJS.jl
